# CroweHello

This is a "Hello World" style project for Crowe-Horwath.

## Requirements

Write a "Hello World" program. 

- [x] The program has 1 current business requirement – write "Hello World" to the console/screen. 
- [x] The program should have an API that is separated from the program logic to eventually support mobile applications, web applications, or console applications, or windows services. 
- [x] The program should support future enhancements for writing to a database, console application, etc. 
- [x] Use common design patterns (inheritance, e.g.) to account for these future concerns. 
- [x] Use configuration files or another industry-standard mechanism for determining where to write the information to. 

Write unit tests to support the API.
